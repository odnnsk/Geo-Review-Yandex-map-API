## Getting started
 - npm: `npm install`
 - yarn: `yarn install`
 
 
 ## Commands
 * `npm run start` - start webpack dev server
 * `npm run dev` - build project dev mode
 * `npm run build` - build project
 * `npm run eslint` - codestyle
